If the driver is not installed, the Nano board will not be able to be recognized by your computer. Therefore, before using it, please install appropriate driver. 
For Windows users, run PL2303_Prolific_DriverInstaller_v1.10.0 in the Driver folder.
For Mac users, refer to the PL2303_MacOSX_10_6-10_10_v1_5_1 in the Driver folder.
